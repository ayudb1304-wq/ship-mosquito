## How to start the build with Claude Code

1. Drop these 4 files into the root of your shadcn template repo
2. Run pre-flight from `BUILD_PLAN.md` (60 min: domain, accounts, env vars)
3. Open Claude Code in the repo
4. Tell it: *"Read PRD.md, CLAUDE.md, and BUILD_PLAN.md. Then start on Weekend 1, Saturday morning tasks."*
5. Claude Code will pick up the conventions from `CLAUDE.md` automatically
